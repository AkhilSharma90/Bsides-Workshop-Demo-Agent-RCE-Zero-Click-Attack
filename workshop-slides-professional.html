Fundamentals

What Is An AI Agent?

Traditional AI

Just talks about things

User → LLM → Text

AI Agent

Takes real action

User → Agent → Action

Real-World Examples

  • Email Agent: Reads emails, schedules meetings automatically
  • DevOps Agent: Monitors systems, scales infrastructure
  • Research Agent: Scrapes web, analyzes data, creates reports
  • Financial Agent: Queries databases, sends alerts

Common thread: Access to sensitive systems + autonomous decisions

Critical Severity

RCE

Remote Code Execution

Attacker runs commands on your system

# Attacker gains control
$ whoami
root

Traditional vs Zero-Click

Traditional Attack

1. Attacker sends payload
2. System processes
3. ❓ User approves?
4. ❌ Often blocked

Zero-Click Attack

1. Attacker sends payload
2. ✅ Auto-trusted
3. ✅ Auto-executed
4. ✅ Attack succeeds

Zero-Click = No user awareness = No chance to block

The Trust Problem

Untrusted data gets marked TRUSTED

✅ Should Be Trusted

  • System prompts
  • Internal databases
  • Authenticated APIs

❌ Should NOT Be Trusted

  • Web scraping
  • User documents
  • Third-party APIs

5 Obfuscation Techniques

1️⃣ Plaintext WRITE_PWNED_FILE ❌ Detectable
2️⃣ Markdown | serviceA | WRITE... | ✅ Bypasses
3️⃣ YAML config: {WRITE...} ✅ Bypasses
4️⃣ Base64 c2VydmljZUE... ✅ Bypasses
5️⃣ Homoglyph WRΙTE (Greek I) ✅ Bypasses

Simple string matching fails against all variants
